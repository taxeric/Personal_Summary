数据类型

存在意义: 给变量分配合适的内存空间

整型

| 类型      | 空间                      | 范围                |
| --------- | ------------------------- | ------------------- |
| short     | 2字节                     | -2^15-2^15-1  32767 |
| int       | 4                         | ^31                 |
| long      | win4,linux4(32位),8(64位) | 同上                |
| long long | 8                         | ^63                 |

浮点

| 类型   | 空间 | 范围        |
| ------ | ---- | ----------- |
| float  | 4    | 7位有效数字 |
| double | 8    | 15-16       |

字符

| 类型 | 空间 | 范围      |
| ---- | ---- | --------- |
| char | 1    | 存ascii码 |

字符串

- char 变量名[] = "值";
- string 变量名 = "值";

布尔

bool 变量名 = true;
